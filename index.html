A bit about me

I have AuDHD (autism and ADHD), which means I dive deep into the things I love. Right now that means teaching myself web development — I built this entire site to practise HTML, CSS, and a bit of JavaScript.

I'm also learning C# and plan to pick up Java next. This portfolio is my living project — it'll keep growing as I do.
